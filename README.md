Установка Arch linux
=====================

### [BIOS установка]
* pacman -Sy && pacman -S wget
* wget raw.githubusercontent.com/kronsky/archlinux/main/bios-install.sh
* ./install.sh

### [EFI установка]
* pacman -Sy && pacman -S wget
* wget raw.githubusercontent.com/kronsky/archlinux/main/efi-install.sh
* ./efi-install.sh
