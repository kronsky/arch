Установка Arch linux
=====================

### [BIOS установка]
* pacman -Sy && pacman -S wget
* wget https://raw.githubusercontent.com/kronsky/arch/main/bios-install.sh
* ./install.sh

### [EFI установка]
* pacman -Sy && pacman -S wget
* wget https://raw.githubusercontent.com/kronsky/arch/main/efi-install.sh
* ./efi-install.sh
